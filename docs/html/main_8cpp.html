◆ handleOption()

void handleOption ( int option)

Processes the selected option from the main menu.

Based on the user's input, this function calls the appropriate route calculation function or exits the application.

Parameters
optionThe menu option selected by the user.
  • 1: Calculate the fastest route.
  • 2: Calculate an independent second-fastest route.
  • 3: Calculate a route with restricted nodes/segments.
  • 4: Calculate an eco-friendly route.
  • 5: Execute batch processing from input.txt.
  • 6: Exit the application.
Note
Time Complexity: O(n), where n is the number of menu options, as it involves checking the selected option and performing corresponding actions.
