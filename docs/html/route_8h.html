Finds an eco-friendly route balancing walking and driving.

Computes a route that minimizes driving while considering a maximum walking time and avoiding specific nodes and segments.

Parameters
gThe graph in which to find the route.
sourceThe starting node.
destThe destination node.
maxWalkTimeThe maximum allowable walking time.
avoidNodesA set of nodes to avoid.
avoidSegmentsA set of edges to avoid.
messageA reference string to store messages about the route calculation.
Returns
A tuple containing the walking path, a message, and the driving path.
Note
Time Complexity: O((E + V) * log V + P * (E + V)), where E is the number of edges, V is the number of vertices, and P is the number of parking candidates. The complexity comes from running Dijkstra's algorithm multiple times for each parking candidate.
