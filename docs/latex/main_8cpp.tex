Processes the selected option from the main menu. 

Based on the user\textquotesingle{}s input, this function calls the appropriate route calculation function or exits the application.


\begin{DoxyParams}{Parameters}
{\em option} & The menu option selected by the user.
\begin{DoxyItemize}
\item 1\+: Calculate the fastest route.
\item 2\+: Calculate an independent second-\/fastest route.
\item 3\+: Calculate a route with restricted nodes/segments.
\item 4\+: Calculate an eco-\/friendly route.
\item 5\+: Execute batch processing from input.\+txt.
\item 6\+: Exit the application.
\end{DoxyItemize}\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
Time Complexity\+: O(n), where n is the number of menu options, as it involves checking the selected option and performing corresponding actions.
